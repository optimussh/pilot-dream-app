# Pilot Dream 쉬운 가이드 (초등학생용)

이 문서는 **새로 추가된 기능을 아이 눈높이**로 설명해요.  
앱 안 **📖 가이드** 메뉴에서도 같은 내용을 볼 수 있어요.

---

## 🏢 우리 회사 손익 보드 (2층 — 회사 경영)

**어디에 있어요?**  
`내 항공사` → `📊 개요` 맨 위 카드들 아래

### 세 가지 숫자

| 이름 | 쉬운 말 | 예시 |
|------|---------|------|
| **매출** | 들어온 돈 | 비행 손님, 화물, 부가서비스 |
| **비용** | 쓴 돈 | 직원 주급 |
| **이익** | 남는 돈 | 매출 − 비용 |

> 💡 **이익이 플러스(+)** 면 잘 버는 중!  
> **마이너스(−)** 면 쓰는 돈이 더 많은 거예요. 노선을 키우거나 주급을 조절해 보세요.

### CEO 회의 (이익 배치) — 주 1번

이익이 생기면 **이번 주 한 번** 방향을 고를 수 있어요.

1. 🏦 **금고에 보관**  
   - 지갑 돈을 **회사 금고**로 옮겨 모아요.  
   - 나중에 큰 비행기를 살 때 쓸 수 있어요.  
   - 개념: **저축**

2. 📈 **성장에 재투자**  
   - 돈을 다시 회사에 넣는 선택이에요.  
   - 이번 주 **노선 매출이 조금 올라요 (+6%)**.  
   - 개념: **재투자 · 성장**

3. 🎁 **직원 보너스**  
   - 동료에게 용돈을 나눠 줘요.  
   - **평판**이 올라가요.  
   - 개념: **사람 투자**

### 회사 금고

- 금고에 모은 돈은 개요 보드 오른쪽 위에서 볼 수 있어요.
- **금고→지갑** 버튼으로 다시 꺼낼 수 있어요.

### 왜 배우나요?

진짜 회사 CEO도  
“남은 돈을 **모아둘지 / 회사에 다시 넣을지 / 사람과 나눌지**”  
를 고민해요.  
이게 나중에 **주식회사·투자**를 이해하는 첫걸음이에요!

## 📈 투자 교실 (3층 — 주식회사·투자)

**어디에 있어요?**  
`내 항공사` → `📈 투자` 탭

### 무엇을 배우나요?

| 쉬운 말 | 의미 |
|---------|------|
| 회사 조각 | 주식처럼 회사를 나누는 몫 |
| 조각 주인 | 주주 |
| 배당 | 잘되면 돌아오는 용돈 |
| 투자 | 나중에 도움이 되도록 지금 넣는 돈 |
| 리스크 | 결과가 흔들릴 수 있음 |

### 기능

1. **조각 100개 발행** — 내 몫 100%로 시작  
2. **NPC 투자자** — 거절해도 OK, 수락하면 지분 나눔  
3. **하늘 친구 시장** — 가상 회사 6곳, **하루 2번**만  
4. **단타 금지** — 산 주에는 못 팔아요  
5. **7일 인내 보너스** — 오래 가지면 작은 보너스  
6. **주간 배당** · **이사회 주 1카드**

### 중독 방지 (부모님께)

- 시세·배당·이사회는 **주 단위**  
- 실시간 차트·빚·레버리지 없음  
- 비행·학습·노선이 메인, 투자는 부록
